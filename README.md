# 📚 英语绘本 Demo — "I Want a Party!"

> **Demo 的核心目标**：探索一种有趣的儿童英语绘本交互形式。用最简单的制作成本（单页 HTML + 素材图片），验证可以规模化扩充的绘本内容方案。

## 🎯 为什么做这个 Demo

传统英语绘本的交互方式单一，孩子容易失去兴趣。我们希望验证一个假设：

**通过精巧的互动机制 + 游戏化反馈，简单的图文素材也能创造出让孩子主动探索的学习体验。**

如果这套交互模式被验证有效，后续可以低成本地规模化扩充绘本内容——只需替换图片和文本数据，无需修改代码。

## 🏗️ 技术实现

- **纯原生技术栈** — 单文件 HTML + CSS3 + Vanilla JavaScript，零框架依赖
- **Web Speech API** — 浏览器原生语音合成
- **Web Audio API** — Animalese 风格合成音效
- **CSS 动画** — 弹性动画、翻页过渡、confetti 粒子效果
- **触摸手势** — touchstart/touchmove/touchend 实现滑动翻页

## 🚀 规模化扩充方案

当前 Demo 验证了核心交互模式，后续扩充绘本内容只需：

1. **准备素材** — 新绘本的页面图片（WebP 格式）+ 背景音乐
2. **编写数据** — 在 `PAGES` 数组中新增页面数据（图片路径、英文文本、单词列表、热点坐标）
3. **补充翻译** — 在 `TRANSLATIONS` 对象中添加新单词的中文翻译，并在 `PAGES` 中填写整句 `translation`

无需修改任何交互逻辑代码，即可快速生成新的互动绘本。

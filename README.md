# 📚 英语绘本 Demo

一个简易而有趣的互动英语绘本体验，专为儿童探索学习设计。

## 项目简介

这个 Demo 旨在探索如何用最简化的生产资源，构建一个既不失趣味性、又能鼓励孩子主动探索的绘本阅读体验。

我们相信，好的儿童内容不需要复杂的制作流程——通过巧妙的设计和互动机制，简单的素材也能创造出引人入胜的学习体验。

## ✨ 功能特点

- **互动式阅读** — 点击页面上的热点，探索隐藏的单词和知识卡片
- **双语支持** — 一键切换中英文翻译，帮助理解
- **语音朗读** — 支持两种语音模式：
  - 🐾 Animalese 风格音效（游戏化语音）
  - 🔊 本地系统语音
- **自动翻页** — 探索完当前页后自动进入下一页
- **背景音乐** — 轻快的背景音乐营造阅读氛围
- **主题切换** — 支持黄色/紫色两种配色主题
- **庆祝动画** — 完成页面探索后会有 confetti 庆祝效果

## 🎮 操作指南

| 操作 | 说明 |
|------|------|
| 点击喇叭按钮 | 朗读当前页面文字 |
| 点击热点 | 探索单词，显示知识卡片 |
| 点击知识卡片 | 朗读单词发音 |
| 左右箭头 | 切换上一页/下一页 |
| 设置按钮 (⚙️) | 切换主题和语音模式 |
| 空格键 | 朗读当前页面 |
| 方向键 | 翻页 |
| M 键 | 切换背景音乐 |
| ESC 键 | 重新开始阅读 |

## 🎨 设计规范

### 色彩系统

| 变量名 | 值 | 用途 |
|--------|-----|------|
| `--yellow-1` | `#FCF9DE` | 背景色 |
| `--yellow-4` | `#FCDA00` | 主色调/强调色 |
| `--orange-4` | `#FF8C00` | 按钮激活状态 |
| `--orange-5` | `#E35B00` | 高亮文字 |
| `--gray-6` | `#404040` | 主要文字 |
| `--gray-4` | `#B2B2B2` | 次要文字 |
| `--green-4` | `#5DC43B` | 成功/正面 |
| `--brown-3` | `#E0BF94` | 装饰边框 |

### 字体

- **JOJO ABC** — 英文标题和正文
- **JOJO ABC Ecard** — 特殊图标字符
- **PingFang SC** — 中文内容

### 动效规范

- **弹性动画**: `cubic-bezier(0.34, 1.56, 0.64, 1)` — 用于弹出、弹跳效果
- **标准动画**: `cubic-bezier(0.22, 1, 0.36, 1)` — 用于页面过渡

## 🛠️ 技术栈

### 核心技术

- **纯原生技术栈** — HTML5 + CSS3 + Vanilla JavaScript
- **无框架依赖** — 轻量级实现，快速加载
- **Web Speech API** — 系统语音合成
- **Web Audio API** — Animalese 音效引擎

### 第三方资源

| 资源 | 用途 |
|------|------|
| JOJO ABC 字体家族 | 儿童友好的英文字体 |
| Nintendo Sound Team - New Years Eve | 背景音乐 |

### 浏览器兼容性

- Chrome / Edge / Safari / Firefox 最新版
- 支持移动端触摸操作
- 响应式设计，适配各种屏幕尺寸
